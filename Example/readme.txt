Execução de Exemplo

• Executando
Para executar o exemplo basta executar num terminal o comando de compilação de código Cobol.
$   cobc -xj COBOLINIT.cbl -I ./COPY

• Argumentos
Abaixo estão alguns dos argumentos que podem ser adicionados no terminal para compilar o código Cobol.
-I <diretório>
   Adicione um <diretório> para copiar/incluir na pesquisa para execução. 
-L <diretório>
   Adicione um <diretório> ao caminho de pesquisa da biblioteca. 
-l <lib>
   Link da biblioteca <lib>. 
-D <define>
   Passar <define> para o compilador COBOL. 
-o <arquivo>
   Coloque a saída em <file>.
